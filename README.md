# Calebcauthon's Homebrew Tap

This is a [Homebrew](https://brew.sh) tap containing formulae for my projects.

## Installation

```bash
brew tap calebcauthon/tap
brew install agent-tee
```

Or install directly:

```bash
brew install calebcauthon/tap/agent-tee
```

## Available Formulae

- **agent-tee** - Run commands while teeing stdout+stderr to per-concern log files
